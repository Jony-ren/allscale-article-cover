# Prompt Examples

## Generate From URL

```text
Use $allscale-article-cover 生成封面图：
https://www.allscale.io/posts/smart-contracts-business-payments
```

## Generate And Explain

```text
Use $allscale-article-cover 生成封面图：
https://www.allscale.io/posts/ai-agents-stablecoin-payments-autonomous-financial-automation

并且告诉我为什么这么生成。
```

## Cover Direction Only

```text
Use $allscale-article-cover 先不要生图。
请帮我为这篇文章设计封面方向：
- 两行标题
- 右上角 object-pack 图标
- 左下角 object-pack 图标
- 右下角照片卡建议
- 为什么这么选

<paste article text>
```

## Regenerate With Layout Constraints

```text
Use $allscale-article-cover 重新生成这篇文章封面。
要求：
- 标题严格两行
- 标题 line-height 等于字号
- AllScale logo 宽度约等于 2.5 个标题字母宽
- 右下角图片右边距和下边距相等
- 右下角图片不能露出绿色边缘
```
